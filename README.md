## üöÄ Quick Start

### 1. Prerequisites

- Node.js (v20+)
- Python (v3.10+)

### 2. Backend Setup

```bash
cd backend
pip install -r requirements.txt
python -m uvicorn main:app --reload
```

Set up your `.env` in the root:

```env
DEEPSEEK_API_KEY=your_key_here
```

### 3. Frontend Setup

```bash
# Install dependencies
npm install

# Start development mode
npm run dev
```
